Adwaita-ish Dark Syntax Theme
=============================

A syntax theme to resemble Adwaita Dark for the [Atom editor](https://atom.io).
Works best with [adwaita-ish-dark-ui](https://github.com/73/adwaita-ish-dark-ui).
